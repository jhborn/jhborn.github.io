HTML 1.0

The original version of HTML was HTML 1.0. It had very limited features which greatly limited what you could do in designing your web pages.

HTML 2.0

HTML 2.0 then arrived and included all the features of HTML 1.0 plus several new features for web page design. Until January, 1997, HTML 2.0 was the standard in web page design.

HTML 3.0

a proposed, ambitious update to HTML that introduced many features now common on the web, such as tables, text flow around figures, and mathematical equations, but it was never officially released

HTML 3.2

HTML 3.2 captures the recomended practice as of early 1996 and became the official standard in january 1997 most if not all popular browsers in use today fully support HTML 3.2

HTML 4.0

Was code named COUGAR. this version intrroduces new functionality, most comes from an expired 3.0 draft. became the recomendation in december 1997

HTML 4.01

a web standard that was a significant upgrade to the Hypertext Markup Language, introduced in 1999. It improved on previous versions by adding features for style sheets, scripting, better printing, and accessibility, and it supported more multimedia and internationalization.

XHTML 1.0

designed to help techknowlodgy access the internet and for people with disabilities.

HTML5

. HTML5 is the result of cooperation that began in 2006 between the World Wide Web Consortium (W3C) and the Web Hypertext Application Technology Working Group (WHATWG)
